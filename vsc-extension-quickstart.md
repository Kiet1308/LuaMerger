# Welcome to your VS Code Extension

## What's in the folder
- `package.json` - manifest file with extension metadata and command contribution.
- `extension.js` - extension entry point where activation/cleanup happen.
- `.vscode/launch.json` - configuration to run the extension in the Extension Development Host.
- `.vscode/tasks.json` - sample task to run `npm test`.
- `.vscodeignore` - files excluded from the packaged extension.

## Get up and running straight away
- Press `F5` to open a new Extension Development Host window.
- Open the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P` on macOS) and run `Lua Merger: Hello World`.

## Make changes
- You can reload the Extension Development Host after edits with `Ctrl+R` (or `Cmd+R` on macOS).

## Explore the API
- You can find the full VS Code API docs at https://code.visualstudio.com/api.

## Run tests
- Update `npm test` and the sample task in `.vscode/tasks.json` to add real tests.
